# fspez

Flutter Reddit client — cookie-based auth, no OAuth required.

## Features

- **Cookie-based login** via WebView (no Reddit API token needed)
- **Feed**: Home, Popular, and sortable Popular feed with hot/new/top/rising/controversial
- **Pull-to-refresh** on all feeds (standard RefreshIndicator)
- **Vote, save, hide, delete** posts and comments with optimistic UI updates
- **Open link posts** in in-app browser
- **Collapsible threaded comments** with vote/save/reply/delete
- **Post submission** (text and link)
- **User profiles** with Posts / Comments / About tabs
- **Multi-account** support with account switcher; My Profile, Saved, and Hidden screens
- **Inbox** with All / Unread / Sent tabs, expandable message threads, inline replies
- **Private messages** via HTTP compose
- **Search** with infinite scroll
- **Dark mode** (follows system theme)
